Extensión inactiva

🔄 Toggle Test - Image Format Detector

📋 Instrucciones para probar la activación/desactivación:

  1. Estado inicial: La extensión inicia desactivada (no invasiva)
  2. Activar: Haz clic en el icono de la extensión en la barra de herramientas
  3. Observar: Las imágenes se marcarán automáticamente con bordes y etiquetas
  4. Desactivar: Haz clic nuevamente en el icono para desactivar
  5. Verificar: Los bordes desaparecerán y las imágenes volverán a su estado original

🎮 Controles de demostración

Indicador de estado: El indicador en la esquina superior derecha muestra si la extensión está activa o inactiva.

Consola de desarrollador: Abre F12 para ver logs detallados de activación/desactivación.

Popup: También puedes usar el popup de la extensión para activar/desactivar.

🖼️ Imágenes de prueba

Estas imágenes se marcarán automáticamente cuando actives la extensión:

Imagen PNG
Imagen JPG
Imagen GIF
Imagen SVG

🔄 Comportamiento esperado

  • Al activar: Todas las imágenes se marcarán con bordes negros y etiquetas de formato
  • Al desactivar: Los bordes desaparecerán y las imágenes volverán a su estado original
  • Nuevas imágenes: Si se añaden imágenes dinámicamente, solo se marcarán si la extensión está activa
  • Persistencia: El estado se mantiene al navegar entre páginas

🔍 Información de debug

Abre las herramientas de desarrollador (F12) para ver logs detallados:

  • 🟢 Image Format Detector activado - Cuando se activa la extensión
  • 🔴 Image Format Detector desactivado - Cuando se desactiva la extensión
  • Image URL: ... - URLs de las imágenes procesadas
  • Detected extension: ... - Extensiones detectadas
  • Format: ... - Formatos finales detectados
