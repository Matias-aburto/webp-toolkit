📏 Dimensions Test - WebP Toolkit

📋 Instrucciones para probar las dimensiones máximas:

  1. Abre la extensión: Haz clic en el icono de la extensión
  2. Ve a la pestaña "Convertir imágenes"
  3. Configura dimensiones máximas: Usa los inputs de ancho y alto máximo
  4. Arrastra imágenes: Prueba con imágenes de diferentes tamaños
  5. Verifica el resultado: Las imágenes se redimensionarán manteniendo la proporción

💡 Información sobre dimensiones máximas

Ancho máximo: Limita el ancho de la imagen. Si solo se especifica ancho, el alto se ajusta automáticamente.

Alto máximo: Limita el alto de la imagen. Si solo se especifica alto, el ancho se ajusta automáticamente.

Ambos valores: Si se especifican ambos, la imagen se ajusta para no exceder ninguna de las dos dimensiones.

Proporción de aspecto: Siempre se mantiene la proporción original de la imagen.

🎯 Casos de prueba sugeridos

  • Ancho máximo 500px: Todas las imágenes tendrán máximo 500px de ancho
  • Alto máximo 300px: Todas las imágenes tendrán máximo 300px de alto
  • Ancho 800px, Alto 600px: Las imágenes se ajustarán para no exceder ninguna dimensión
  • Sin límites: Las imágenes mantienen su tamaño original

📊 Beneficios del redimensionamiento

  • Optimización de tamaño: Imágenes más pequeñas = archivos WebP más pequeños
  • Mejor rendimiento: Carga más rápida en sitios web
  • Consistencia: Todas las imágenes tienen dimensiones uniformes
  • Flexibilidad: Puedes especificar solo ancho, solo alto, o ambos
  • Proporción mantenida: No se distorsiona la imagen

🔧 Cómo usar la funcionalidad

1. Abre la extensión WebP Toolkit

2. Ve a la pestaña "🔄 Convertir imágenes"

3. En la sección "Dimensiones máximas", ingresa los valores deseados

4. Arrastra las imágenes que quieres convertir

5. Haz clic en "Convertir todas a WebP"

6. Las imágenes se descargarán con las nuevas dimensiones
