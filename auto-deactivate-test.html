Extensión inactiva

🔄 Auto-Deactivate Test - Image Format Detector

📋 Instrucciones para probar la auto-desactivación:

  1. Activa la extensión: Haz clic en el icono de la extensión
  2. Observa los bordes: Las imágenes se marcarán con bordes delgados
  3. Cambia de página: Usa los enlaces de navegación de abajo
  4. Verifica la desactivación: La extensión se desactivará automáticamente
  5. Reactiva manualmente: Debes activar la extensión nuevamente

🎮 Controles de demostración

Indicador de estado: El indicador en la esquina superior derecha muestra si la extensión está activa o inactiva.

Navegación: Usa los enlaces de abajo para simular navegación entre páginas.

Consola de desarrollador: Abre F12 para ver logs detallados de activación/desactivación.

💡 Comportamiento esperado

Activación manual: La extensión solo se activa cuando tú lo decides.

Desactivación automática: Se desactiva automáticamente al cambiar de página, pestaña o URL.

Bordes delgados: Los bordes ahora son de 5px en lugar de 10px para mejor experiencia visual.

Control total: Tú tienes control completo sobre cuándo usar la extensión.

🔍 Información de debug

Abre las herramientas de desarrollador (F12) para ver logs detallados:

  • 🟢 Image Format Detector activado - Cuando se activa la extensión
  • 🔴 Image Format Detector desactivado - Cuando se desactiva manualmente
  • 🔴 Image Format Detector desactivado automáticamente al cambiar de página - Auto-desactivación
  • Bordes delgados (5px) - Nuevo grosor de borde

📊 Beneficios de la auto-desactivación

  • Menos invasiva: No interfiere con la navegación normal
  • Control total: Solo funciona cuando tú lo decides
  • Mejor rendimiento: No consume recursos innecesariamente
  • Experiencia limpia: Los bordes desaparecen automáticamente
  • Bordes delgados: Mejor experiencia visual
